# Retail-Sales-Profit-Analysis-using-SQL
Project Overview 

This project analyzes retail sales, profit, and marketing data using SQL to uncover key business insights. The objective was to identify high-performing products, evaluate regional sales performance, and understand the impact of cost and marketing on profitability.

Using structured querying and advanced SQL techniques, the analysis highlights patterns in sales trends, cost drivers, and operational efficiency to support data-driven decision making.

Dataset Description 

The analysis is based on three relational datasets:

Fact Table: 4,200+ transaction records including sales, profit, COGS, and expenses

Product Table: Product categories and types

Location Table: Geographic data including state and market size
